Affine Sets

Affine Set: Affine sets CCC are the sets which contains the line through any two distinct points in CCC. That is, for any x1,x2∈C⊆Rnx_1, x_2\in C \subseteq \mathbf{R}^nx1​,x2​∈C⊆Rn and θ∈R\theta \in \mathbf{R}θ∈R, we have θx1+(1−θ)x2∈C\theta x_1+(1-\theta)x_2 \in Cθx1​+(1−θ)x2​∈C.

Affine Combination: Affine combination of x1,...,xkx_1, ..., x_kx1​,...,xk​ is given by θ1x1+...+θkxk\theta_1x_1+...+\theta_kx_kθ1​x1​+...+θk​xk​ where θ1+...+θk=1\theta_1+...+\theta_k=1θ1​+...+θk​=1.

If CCC is an affine set, and x1,...,xk∈Cx_1, ..., x_k\in Cx1​,...,xk​∈C, their affine combination belongs to CCC.

Example: The solution set of a system of linear equations C={x ∣ Ax=b}C=\{x\ |\ Ax=b\}C={x ∣ Ax=b}, where A∈Rm×nA\in \mathbf{R}^{m\times n}A∈Rm×n and b∈Rmb\in \mathbf{R}^mb∈Rm, is an affine set. Conversely, every affine set can be expressed as the solution set of a system of linear equations.

Affine Hull: Affine hull of some set C⊆RnC\subseteq \mathbf{R}^nC⊆Rn is the set of all affine combinations of points in CCC, denoted by

aff C={θ1x1+...+θkxk ∣ x1,...,xk∈C,θ1+...+θ+k=1}\mathbf{aff}\ C=\{\theta_1x_1+...+\theta_kx_k\ |\ x_1,...,x_k \in C, \theta_1+...+\theta+k=1\} aff C={θ1​x1​+...+θk​xk​ ∣ x1​,...,xk​∈C,θ1​+...+θ+k=1}

The affine hull is the smallest affine set that contains CCC.
