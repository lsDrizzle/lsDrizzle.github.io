Affine and Convex Set

Lines

Suppose x1≠x2x_1\ne x_2x1​​=x2​ are two points in Rn\mathbf{R}^nRn, and θ∈R\theta \in \mathbf{R}θ∈R. The line passing through them is given by y=θx1+(1−θ)x2y=\theta x_1+(1-\theta)x_2y=θx1​+(1−θ)x2​.

Affine Sets

Affine Set: Affine sets CCC are the sets which contains the line through any two distinct points in CCC. That is, for any x1,x2∈C⊆Rnx_1, x_2\in C \subseteq \mathbf{R}^nx1​,x2​∈C⊆Rn and θ∈R\theta \in \mathbf{R}θ∈R, we have θx1+(1−θ)x2∈C\theta x_1+(1-\theta)x_2 \in Cθx1​+(1−θ)x2​∈C.

Affine Combination: Affine combination of x1,...,xkx_1, ..., x_kx1​,...,xk​ is given by θ1x1+...+θkxk\theta_1x_1+...+\theta_kx_kθ1​x1​+...+θk​xk​ where θ1+...+θk=1\theta_1+...+\theta_k=1θ1​+...+θk​=1.

If CCC is an affine set, and x1,...,xk∈Cx_1, ..., x_k\in Cx1​,...,xk​∈C, their affine combination belongs to CCC.

Example: The solution set of a system of linear equations C={x ∣ Ax=b}C=\{x\ |\ Ax=b\}C={x ∣ Ax=b}, where A∈Rm×nA\in \mathbf{R}^{m\times n}A∈Rm×n and b∈Rmb\in \mathbf{R}^mb∈Rm, is an affine set. Conversely, every affine set can be expressed as the solution set of a system of linear equations.

Affine Hull: Affine hull of some set C⊆RnC\subseteq \mathbf{R}^nC⊆Rn is the set of all affine combinations of points in CCC, denoted by

aff C={θ1x1+...+θkxk ∣ x1,...,xk∈C,θ1+...+θ+k=1}\mathbf{aff}\ C=\{\theta_1x_1+...+\theta_kx_k\ |\ x_1,...,x_k \in C, \theta_1+...+\theta+k=1\} aff C={θ1​x1​+...+θk​xk​ ∣ x1​,...,xk​∈C,θ1​+...+θ+k=1}

The affine hull is the smallest affine set that contains CCC.

Affine Dimension and Relative Interior

Affine Dimension: Affine dimension of a set CCC is the dimension of its affine hull.

Sometimes, the affine dimension of a set C⊆RnC\subseteq \mathbf{R^n}C⊆Rn is less than nnn. For example, a square in the (x1,x2)(x_1,x_2)(x1​,x2​)-plane in R3\mathbf{R}^3R3 is defined as

C={x∈R3 ∣ −1⩽x1⩽1, −1⩽x2⩽1, x3=0}C=\{x\in \mathbf{R}^3\ |\ -1\leqslant x_1\leqslant 1,\ -1\leqslant x_2\leqslant 1,\ x_3=0\} C={x∈R3 ∣ −1⩽x1​⩽1, −1⩽x2​⩽1, x3​=0}

The interior of CCC is empty. But its affine hull aff C={x∈R3 ∣ x3=0}\mathbf{aff}\ C=\{x\in \mathbf{R}^3\ |\ x_3=0\}aff C={x∈R3 ∣ x3​=0}, so its interior relative to aff C\mathbf{aff}\ Caff C can be regarded as

relint C={x∈R3 ∣ −1<x1<1, −1<x2<1, x3=0}\mathbf{relint}\ C=\{x\in \mathbf{R}^3\ |\ -1<x_1<1,\ -1<x_2<1,\ x_3=0\} relint C={x∈R3 ∣ −1<x1​<1, −1<x2​<1, x3​=0}

Its boundary is itself. But its boundary relative to aff C\mathbf{aff}\ Caff C can be regarded as

{x∈R3 ∣ max{∣x1∣, ∣x2∣}=1, x3=0}\{x\in \mathbf{R}^3\ |\ \mathbf{max}\{|x_1|,\ |x_2|\}=1,\ x_3=0\} {x∈R3 ∣ max{∣x1​∣, ∣x2​∣}=1, x3​=0}

Relative Interior: The relative interior of a set CCC is denoted by

relint C={x∈C ∣ ∃ r>0, B(x,r)∩aff C⊆C}\mathbf{relint}\ C=\{x\in C\ |\ \exists \ r>0,\ B(x,r)\cap \mathbf{aff}\ C\subseteq C\} relint C={x∈C ∣ ∃ r>0, B(x,r)∩aff C⊆C}

where B(x,r)={y ∣ ∣∣y−x∣∣⩽r}B(x,r)=\{y\ |\ ||y-x||\leqslant r\}B(x,r)={y ∣ ∣∣y−x∣∣⩽r} is the ball of radius rrr and center xxx in the norm ∣∣⋅∣∣||·||∣∣⋅∣∣ (can be any norm; all norms define the same relative interior).

Relative Boundary: The relative boundary of a set CCC is defined as cl C \ relint C\mathbf{cl}\ C\ \backslash\ \mathbf{relint}\ Ccl C \ relint C, where cl C\mathbf{cl}\ Ccl C is the closure of CCC.

Convex Set

Convex Set: A set CCC is convex if the line segment between any two points in CCC lies in CCC, i.e., ∀x1, x2∈C,∀θ∈[0,1]\forall x_1,\ x_2\in C, \forall \theta\in[0,1]∀x1​, x2​∈C,∀θ∈[0,1], we have θx1+(1−θ)x2∈C\theta x_1+(1-\theta)x_2\in Cθx1​+(1−θ)x2​∈C.

Every affine set is convex.

Convex Combination: Convex combination of x1,...,xkx_1, ..., x_kx1​,...,xk​ is given by θ1x1+...+θkxk\theta_1x_1+...+\theta_kx_kθ1​x1​+...+θk​xk​ where θ1+...+θk=1\theta_1+...+\theta_k=1θ1​+...+θk​=1 and θi⩾0\theta_i\geqslant0θi​⩾0 for i=1,...,ki=1,...,ki=1,...,k.

If CCC is a convex set, and x1,...,xk∈Cx_1, ..., x_k\in Cx1​,...,xk​∈C, their convex combination belongs to CCC. A convex combination can be thought of as a mixture or weighted average of those points.

Convex Hull: The convex hull of a set C⊆RnC\subseteq \mathbf{R}^nC⊆Rn is the set of all convex combinations of points in CCC, denoted by

conv C={θ1x1+...+θkxk ∣ xi∈C, θi⩾0, i=1,...,k, θ1+...+θk=1}\mathbf{conv}\ C=\{\theta_1x_1+...+\theta_kx_k\ |\ x_i\in C,\ \theta_i\geqslant0,\ i=1,...,k,\ \theta_1+...+\theta_k=1\} conv C={θ1​x1​+...+θk​xk​ ∣ xi​∈C, θi​⩾0, i=1,...,k, θ1​+...+θk​=1}

The convex hull is the smallest convex set that contains CCC.

Cones

Cone: A set CCC is a cone (or nonnegative homogeneous) if ∀x∈C\forall x\in C∀x∈C and ∀θ⩾0\forall\theta\geqslant0∀θ⩾0 we have θx∈C\theta x\in Cθx∈C.

A set is a convex cone if it is convex and a cone, which means ∀x1, x2∈C\forall x_1,\ x_2\in C∀x1​, x2​∈C and ∀θ1, θ2⩾0\forall\theta_1,\ \theta_2\geqslant0∀θ1​, θ2​⩾0, we have θ1x1+θ2x2∈C\theta_1x_1+\theta_2x_2\in Cθ1​x1​+θ2​x2​∈C.

The pie slice shows all points of the form θ1x1+θ2x2\theta_1x_1+\theta_2x_2θ1​x1​+θ2​x2​, where θ1, θ2⩾0\theta_1,\ \theta_2\geqslant0θ1​, θ2​⩾0.

Conic Combination: Conic combination of x1,...,xkx_1, ..., x_kx1​,...,xk​ is given by θ1x1+...+θkxk\theta_1x_1+...+\theta_kx_kθ1​x1​+...+θk​xk​ where θ1,...,θk⩾0\theta_1,...,\theta_k\geqslant0θ1​,...,θk​⩾0 and θi⩾0\theta_i\geqslant0θi​⩾0 for i=1,...,ki=1,...,ki=1,...,k.

If CCC is a convex cone, and x1,...,xk∈Cx_1, ..., x_k\in Cx1​,...,xk​∈C, their conic combination belongs to CCC.

Conic Hull: The conic hull of a set C⊆RnC\subseteq \mathbf{R}^nC⊆Rn is the set of all conic combinations of points in CCC, denoted by

{θ1x1+...+θkxk ∣ xi∈C, θi⩾0, i=1,...,k}\{\theta_1x_1+...+\theta_kx_k\ |\ x_i\in C,\ \theta_i\geqslant0,\ i=1,...,k\} {θ1​x1​+...+θk​xk​ ∣ xi​∈C, θi​⩾0, i=1,...,k}

The conic hull is the smallest convex cone that contains CCC.

Some Important Examples

Hyperplanes and Halfspaces

Hyperplane: A hyperplane is a set of the form {x ∣ aTx=b}\{x\ |\ a^Tx=b\}{x ∣ aTx=b}, where a∈Rn,a≠0a\in \mathbf{R}^n, a\ne0a∈Rn,a​=0 and b∈Rb\in Rb∈R.

It can be interpreted as a hyperplane with normal vector aaa; the constant bbb determines the offset of the hyperplane from the origin. This can be understood by expressing the hyperplane in the form {x ∣ aT(x−x0)=0}=x0+a⊥\{x\ |\ a^T(x-x_0)=0\}=x_0+a^\bot{x ∣ aT(x−x0​)=0}=x0​+a⊥, where x0x_0x0​ is any point in the hyperplane and a⊥={v ∣ aTv=0}a^\bot=\{v\ |\ a^Tv=0\}a⊥={v ∣ aTv=0}. This shows that the hyperplane consists of an offset x0x_0x0​, plus all vectors orthogonal to the normal vector aaa.

Halfspace: A (closed) halfspace is a set of the form {x ∣ aTx⩽b, a≠0}\{x\ |\ a^Tx\leqslant b,\ a\ne 0\}{x ∣ aTx⩽b, a​=0}.

Euclidean Balls and Ellipsoids

Ball: A (Euclidean) ball in Rn\mathbf{R}^nRn has the form

B(xc,r)={x ∣ ∣∣x−xc∣∣2⩽r}={x ∣ (x−xc)T(x−xc)⩽r2}B\left( x_c,r \right) =\left\{ x\,|\, ||x-x_c||_2\leqslant r \right\} =\left\{ x\,| \,\left( x-x_c \right) ^T\left( x-x_c \right) \leqslant r^2 \right\} B(xc​,r)={x∣∣∣x−xc​∣∣2​⩽r}={x∣(x−xc​)T(x−xc​)⩽r2}

where r>0r>0r>0. The vector xcx_cxc​ is the center of the ball and rrr is the radius.

Ellipsoid: An ellipsoid in Rn\mathbf{R}^nRn has the form

E={x ∣ (x−xc)TP−1(x−xc)⩽1}\mathcal{E}=\left\{ x\,| \,\left( x-x_c \right) ^TP^{-1}\left( x-x_c \right) \leqslant 1 \right\} E={x∣(x−xc​)TP−1(x−xc​)⩽1}

where PPP is symmetric and positive definite, i.e., P=PT≻0P=P^T\succ 0P=PT≻0. The lengths of the two semi-axes of E\mathcal{E}E are given by λi\sqrt{\lambda_i}λi​​, where λi\lambda_iλi​ are the eigenvalues of PPP. A ball is an ellipsoid with P=r2IP=r^2IP=r2I.

Another representation of an ellipsoid is

E={xc+Au ∣ ∣∣u∣∣2⩽1}\mathcal{E}=\left\{ x_c+Au\,|\,||u||_2\leqslant 1 \right\} E={xc​+Au∣∣∣u∣∣2​⩽1}

where AAA is square and nonsingular.

Normal Balls and Norm Cones

Suppose ∣∣⋅∣∣||\cdot ||∣∣⋅∣∣ is any norm on Rn\mathbf{R}^nRn.

Norm Ball: A norm ball is defined as {x ∣ ∣∣x−xc∣∣⩽r}\left\{ x\,|\,||x-x_c||\leqslant r \right\}{x∣∣∣x−xc​∣∣⩽r}, and it is convex.

Norm Cone: The norm cone (it is also a convex cone) associated with the norm ∣∣⋅∣∣||\cdot ||∣∣⋅∣∣ is the set

C={(x,t) ∣ ∣∣x∣∣⩽t}⊆Rn+1C=\left\{ \left( x,t \right) \,|\, ||x||\leqslant t \right\} \subseteq \mathbf{R}^{n+1} C={(x,t)∣∣∣x∣∣⩽t}⊆Rn+1

Boundary of second-order cone in R3\mathbf{R}^3R3, {(x1,x2,t) ∣ (x12+x22)1/2⩽t}\left\{ \left( x_1,x_2,t \right) \,|\, \left( x_{1}^{2}+x_{2}^{2} \right) ^{\text{1/}2}\leqslant t \right\}{(x1​,x2​,t)∣(x12​+x22​)1/2⩽t}.

Polyhedra

Polyhedron: A polyhedron is defined as the solution set of a finite number of linear equalities and inequalities:

P={x ∣ ajTx⩽bj, j=1,...,m, cjTx=dj, j=1,...,p}\mathcal{P}=\left\{ x\,| \,a_{j}^{T}x\leqslant b_j, \,j=\text{1,...,}m, \,c_{j}^{T}x=d_j, \,j=\text{1,...,}p \right\} P={x∣ajT​x⩽bj​,j=1,...,m,cjT​x=dj​,j=1,...,p}

Affine se\ts, rays, line segments, and halfspaces are all polyhedra. Polyhedra are convex sets.

It can be written as a compact notation P={x ∣ Ax⪯b,Cx=d}\mathcal{P}=\left\{ x\,|\, Ax\preceq b, Cx=d \right\}P={x∣Ax⪯b,Cx=d}, where

A=[a1T⋮amT],    C=[c1T⋮cpT]A=\left[ \begin{array}{l} a_{1}^{T}\\ \vdots\\ a_{m}^{T}\\ \end{array} \right] , \,\,\,\, C=\left[ \begin{array}{l} c_{1}^{T}\\ \vdots\\ c_{p}^{T}\\ \end{array} \right] A=⎣⎢⎡​a1T​⋮amT​​⎦⎥⎤​,C=⎣⎢⎡​c1T​⋮cpT​​⎦⎥⎤​

and the symbol ⪯\preceq⪯ denotes component-wise inequality (i.e., u⪯vu\preceq vu⪯v means ui⩽viu_i \leqslant v_iui​⩽vi​ for i=1,...,mi=1,...,mi=1,...,m).

Simplexes

Simplex: Suppose the k+1k+1k+1 points v0,...,vk∈Rnv_0,...,v_k\in\mathbf{R}^nv0​,...,vk​∈Rn are affinely independent (i.e., v1−v0,...,vk−v0v_1-v_0,...,v_k-v_0v1​−v0​,...,vk​−v0​ are linearly independent). The simplex determined by them is given by C=conv {v0,...,vk}C=\mathbf{conv}\ \{v_0,...,v_k\}C=conv {v0​,...,vk​}. Its affine dimension is kkk, so it is referred to as a k-dimensional simplex.

A 1-dim simplex is a line segment,; a 2-dim simplex is a triangle (including its interior); a 3-dim simplex is a tetrahedron.

The Positive Semidefinite Cone

We use Sn\mathbf{S}^nSn to denote the set of symmetric n×nn\times nn×n matrices

Sn={X∈Rn×n ∣ X=XT}\mathbf{S}^n=\left\{ X\in \mathbf{R}^{n\times n}\,| \,X=X^T \right\} Sn={X∈Rn×n∣X=XT}

which is a vector space with dimension n(n+1)/2n(n+1)/2n(n+1)/2.

We use S+n\mathbf{S}_{+}^{n}S+n​ to denote the set of symmetric positive semidefinite matrices

S+n={X∈Sn ∣ X⪰0}\mathbf{S}_{+}^{n}=\left\{ X\in \mathbf{S}^n\,|\, X\succeq 0 \right\} S+n​={X∈Sn∣X⪰0}

which means zTXz⩾0z^TXz\geqslant0zTXz⩾0 for all z≠0z\ne0z​=0.

We also use S++n\mathbf{S}_{++}^{n}S++n​ to denote the set of symmetric positive definite matrices

S++n={X∈Sn ∣ X≻0}\mathbf{S}_{++}^{n}=\left\{ X\in \mathbf{S}^n\,|\, X\succ 0 \right\} S++n​={X∈Sn∣X≻0}

S+n\mathbf{S}_{+}^{n}S+n​ is a convex cone.

Operations that preserve convexity

Intersection

The intersection of convex sets are convex.

Example: Let S={x∈Rm ∣ ∣p(t)∣⩽1 ∀∣t∣⩽π/3}S=\{x\in\mathbf{R}^m\ |\ |p(t)|\leqslant1\ \forall |t|\leqslant \pi/3\}S={x∈Rm ∣ ∣p(t)∣⩽1 ∀∣t∣⩽π/3}, where p(t)=∑k=1mxkcos⁡ktp\left( t \right) =\sum_{k=1}^m{x_k\cos kt}p(t)=∑k=1m​xk​coskt. The set SSS can be expressed as the intersection of an infinite number of slabs: S=⋂∣t∣⩽π/3StS=\bigcap_{|t|\leqslant \pi /3}{S_t}S=⋂∣t∣⩽π/3​St​, where

St={x ∣ −1⩽(cos⁡t,...,cos⁡mt)Tx⩽1}S_t=\left\{ x\,| \,-1\leqslant \left( \cos t,...,\cos mt \right) ^Tx\leqslant 1 \right\} St​={x∣−1⩽(cost,...,cosmt)Tx⩽1}

StS_tSt​ is the solution set of a system of linear equations and thereby convex. Hence SSS is convex.

The set SSS for m=2m=2m=2. It is the intersection of an infinite number of slabs.

Affine Functions

Affine Function: A function f:Rn→Rmf: \mathbf{R}^n\rightarrow \mathbf{R}^mf:Rn→Rm is affine if it has the form f(x)=Ax+bf(x)=Ax+bf(x)=Ax+b, where A∈Rm×nA\in\mathbf{R}^{m\times n}A∈Rm×n and b∈Rmb\in\mathbf{R}^mb∈Rm.

The image of a convex set under an affine function is convex, and the inverse image of a convex set is also convex.

The projection of a convex set onto some of its coordinates is convex.

Sum: The sum of two sets is defined as S1+S2={x+y ∣ x∈S1, x2∈S2}S_1+S_2=\{x+y\ |\ x\in S_1,\ x_2\in S_2\}S1​+S2​={x+y ∣ x∈S1​, x2​∈S2​}.

If S1S_1S1​ and S2S_2S2​ are convex, then S1+S2S_1+S_2S1​+S2​ is convex.

Partial Sum: The partial sum of S1,S2∈Rn×RmS_1,S_2\in\mathbf{R}^n\times\mathbf{R}^mS1​,S2​∈Rn×Rm is defined as

S={(x,y1+y2) ∣ (x,y1)∈S1, (x,y2)∈S2}S=\{(x,y_1+y_2)\ |\ (x,y_1)\in S_1,\ (x,y2)\in S_2\} S={(x,y1​+y2​) ∣ (x,y1​)∈S1​, (x,y2)∈S2​}

where x∈Rnx\in\mathbf{R}^nx∈Rn and yi∈Rmy_i\in\mathbf{R}^myi​∈Rm.

For m=0m=0m=0, the partial sum gives the intersection of S1S_1S1​ and S2S_2S2​; for n=0n=0n=0, it is set addition. Partial sums of convex sets are convex.

Linear-Fractional and Perspective Functions

Perspective Function: The perspective function P: Rn+1→RnP:\ \mathbf{R}^{n+1}\rightarrow\mathbf{R}^nP: Rn+1→Rn, with domain dom P=Rn×R++\mathbf{dom}\ P=\mathbf{R}^n\times\mathbf{R_{++}}dom P=Rn×R++​, is P(z,t)=z/tP(z,t)=z/tP(z,t)=z/t. It scales or normalizes vectors so the last component is 1, and them drops the last component.

The perspective function in R3\mathbf{R}^3R3. It contains an opaque horizonal plane x3=0x_3=0x3​=0, with a single pin-hole at the origin, through which light can pass, and a horizontal image plane x3=−1x_3=-1x3​=−1. An object at xxx with x3>0x_3>0x3​>0 forms an image at the point −(x1/x3,x2/x3,1)-(x_1/x_3,x_2/x_3,1)−(x1​/x3​,x2​/x3​,1) on the image plane. The mapping of the position of a source to the position of its image is related to the perspective function.

If C⊆dom PC\subseteq \mathbf{dom}\ PC⊆dom P is convex, them its image P(C)P(C)P(C) is convex. The inverse image of a convex set under the perspective function is also convex.

Linear-Fractional Function: A linear-fractional function is formed by composing the perspective function with an affine function. Suppose g:Rn→Rm+1g: \mathbf{R}^n\rightarrow \mathbf{R}^{m+1}g:Rn→Rm+1 is affine, i.e.,

g(x)=[AcT]x+[bd]g\left( x \right) =\left[ \begin{array}{c} A\\ c^T\\ \end{array} \right] x+\left[ \begin{array}{c} b\\ d\\ \end{array} \right] g(x)=[AcT​]x+[bd​]

where A∈Rm×n, b∈Rm, C∈RnA\in\mathbf{R}^{m\times n},\ b\in\mathbf{R}^m,\ C\in\mathbf{R}^nA∈Rm×n, b∈Rm, C∈Rn and d∈Rd\in\mathbf{R}d∈R. The function g:Rn→Rmg: \mathbf{R}^n\rightarrow \mathbf{R}^{m}g:Rn→Rm given by f=P∘gf=P\circ gf=P∘g, i.e.,

f(x)=(Ax+b)/(cTx+d),    dom f={x ∣ cTx+d>0}f(x)=(Ax+b)/(c^Tx+d),\ \ \ \ \mathbf{dom}\ f=\{x\ |\ c^Tx+d>0\} f(x)=(Ax+b)/(cTx+d),    dom f={x ∣ cTx+d>0}

is called a linear-fractional function.

Linear-fractional functions preserve convexity.

Proper Cones and Generalized Inequalities

Proper Cone: A cone K⊆RnK\subseteq\mathbf{R}^nK⊆Rn is a proper cone if it satisfies the following:

  • KKK is convex.
  • KKK is closed.
  • KKK is solid, which means it has nonempty interior.
  • KKK is pointed, which means it contains no line.
