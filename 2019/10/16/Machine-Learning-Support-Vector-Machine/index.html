Constrained Optimization Problem

The primal problem can be written as

{min⁡x∈Rp  f(x)s.t.mi(x)⩽0,i=1,...,M  nj(x)=0,j=1,...,N\begin{aligned} \begin{cases} \underset{x\in \mathbb{R}^p}{\min}&\,\,f\left( x \right)\\ \text{s}.\text{t}.&m_i\left( x \right) \leqslant 0, i=1,...,M\\ \,\, &n_j\left( x \right) =0, j=1,...,N\\ \end{cases} \end{aligned} ⎩⎪⎪⎨⎪⎪⎧​x∈Rpmin​s.t.​f(x)mi​(x)⩽0,i=1,...,Mnj​(x)=0,j=1,...,N​​

The Lagrange function is L(x,λ,η)=f(x)+∑i=1Mλimi+∑j=1NηjnjL(x,\lambda,\eta)=f(x)+\sum_{i=1}^M\lambda_im_i+\sum_{j=1}^N\eta_jn_jL(x,λ,η)=f(x)+∑i=1M​λi​mi​+∑j=1N​ηj​nj​. Then the primal problem is equivalent to

{min⁡x∈Rp  max⁡λ,η  L(x,λ,η)s.t.  λi⩾0\begin{cases} \underset{x\in \mathbb{R}^p}{\min}\,\,\underset{\lambda , \eta}{\max}\,\,L\left( x,\lambda ,\eta \right)\\ \text{s}.\text{t}.\ \ \lambda _i\geqslant 0\\ \end{cases} ⎩⎨⎧​x∈Rpmin​λ,ηmax​L(x,λ,η)s.t.  λi​⩾0​

which can be regarded as the unconstrained form of the primal problem.

Its dual problem can be written as

{max⁡λ,η  min⁡x∈Rp  L(x,λ,η)s.t.  λi⩾0\begin{cases} \underset{\lambda , \eta}{\max}\,\,\underset{x\in \mathbb{R}^p}{\min}\,\,L\left( x,\lambda ,\eta \right)\\ \text{s}.\text{t}.\ \ \lambda _i\geqslant 0\\\end{cases} ⎩⎨⎧​λ,ηmax​x∈Rpmin​L(x,λ,η)s.t.  λi​⩾0​

The weak duality tells us that the solution of the dual problem is smaller than or equal to that of its primal problem.
